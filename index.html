Welcome to Mastobooster. This Mastodon-Bot boosts every toot which hashtag was given by the admin and is in the federated server list. So, if you toot anything with the provided hashtag, you'll see it in the "Table"- or "Card"-Container - if the server where the Mastobooster-Bot is running, is able to see your toot.
Please be aware, that you'll only see the last 20 toots.
